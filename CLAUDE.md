## Project
- **Name:** `backstage-react-native`
- **What it is:** Migration of an Android bank app to React Native + Re.Pack, plus a "Spotify for Backstage" web platform to create, version, and distribute miniapps consumed by the mobile host as federated remotes or shared packages.
- **Platforms:** Android + iOS (mobile host); Web (Backstage catalog/registry).

## Stack (authoritative — see memory-bank/standards/tech-stack.md)
- **Framework:** React Native (New Architecture).
- **Bundler:** **Re.Pack** (Rspack + Module Federation v2) — **NOT Metro**. Never generate Metro config.
- **Language:** TypeScript (strict).
- **Lists:** FlashList. **Navigation:** native navigators. **State:** `TBD — decide during Inception`.
- **Federation:** host + on-demand remote chunks. Host vs. remote boundaries live in `memory-bank/standards/system-architecture.md`.
- **Monorepo:** pnpm workspaces — miniapps live as workspace packages, consumable as remotes (runtime) or shared packages (build-time).
