# S1.3 — Documentar consumo externo

**Como** miniapp dev, **quiero** saber cómo instalar los `@org/*`, **para** consumirlos desde un repo externo.

## Aceptación
- `.npmrc` de ejemplo (`@org:registry=https://npm.pkg.github.com`, token `read:packages`).
- README/nota con pasos de instalación por versión.

## Notas
- Reusable por el template (Unit 2) y las miniapps.
