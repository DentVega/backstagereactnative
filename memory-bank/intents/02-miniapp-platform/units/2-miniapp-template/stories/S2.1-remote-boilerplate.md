# S2.1 — Boilerplate del remote (expone ./Entry)

**Como** miniapp dev, **quiero** un remote Re.Pack listo, **para** empezar a codear sin configurar federación.

## Aceptación
- `rspack.config.mjs` con `ModuleFederationPluginV2({ name: "__MINIAPP_ID__", exposes: { "./Entry": "./src/Entry.tsx" }, shared: {…singletons…} })`.
- `Entry.tsx` recibe `MiniappEntryProps` + gate de capability; `Screen.tsx` placeholder.
- `manifest.json` (id/version/entry/shared/capabilities) + `react-native.config.js`.
