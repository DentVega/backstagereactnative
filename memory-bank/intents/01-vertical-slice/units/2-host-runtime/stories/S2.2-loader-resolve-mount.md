# S2.2 — host-runtime: resolve + descarga + montaje de remote

**Como** host, **quiero** resolver la URL de una miniapp y montarla en runtime, **para** cargar features on-demand.

## Aceptación
- `resolve(id)` consulta Backstage (`GET /api/resolve`) — desarrollable contra mock.
- Descarga del chunk vía **Re.Pack ScriptManager**; carga `./Entry` del container remoto.
- Monta `<Entry capabilities={...} />` inyectando capabilities scoped.
- Unit tests de `resolve()` y de la lógica de carga (mock del ScriptManager).

## Notas
- Host · sin nativos · integra con Unit 4 (endpoint real) al final.
