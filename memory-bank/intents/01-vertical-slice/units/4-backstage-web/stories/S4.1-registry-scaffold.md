# S4.1 — Scaffold Next.js + registry (register/publish)

**Como** dev de miniapps, **quiero** registrar y publicar miniapps, **para** distribuirlas al host.

## Aceptación
- Repo separado `backstage-web` (Next.js App Router) arranca.
- Instala `@org/miniapp-contract@^x` desde el registry privado (no copia tipos).
- Store del registry simple (JSON en fs o SQLite ligero).
- `POST /api/miniapps` (registrar) y `POST /api/miniapps/:id/publish` (versión + manifest + URL) funcionan.
- Validación del manifest contra el paquete `@org/miniapp-contract`.
- Tests de API (register/publish).

## Notas
- Web (Next.js) · **repo separado** · n/a nativos · sin secretos en respuestas/logs.
