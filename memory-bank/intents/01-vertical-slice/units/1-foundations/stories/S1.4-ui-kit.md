# S1.4 — packages/ui-kit (tokens + primitivas themed)

**Como** dev, **quiero** primitivas themed y tokens, **para** que host y miniapps compartan un sistema visual (light/dark desde el día 1).

## Aceptación
- Tokens en `src/theme/` (`tokens.ts` + `light.ts`/`dark.ts`) por `design-standards.md`.
- `ThemeProvider` + `useTheme()`; primitivas `AppText`, `Box`, `Card`, `Button`.
- Sin hex/valores crudos en componentes; touch targets ≥44×44.
- `tsc` verde; ≥1 test RNTL de una primitiva.

## Notas
- Shared package · sin nativos · Reanimated solo transform/opacity si hay motion.
