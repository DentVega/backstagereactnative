# S1.1 — Monorepo pnpm + workspaces

**Como** equipo, **quiero** un monorepo pnpm con workspaces, **para** que host, miniapps, packages y backstage convivan y compartan dependencias.

## Aceptación
- `pnpm-workspace.yaml` incluye `apps/*`, `miniapps/*`, `packages/*`.
- `pnpm install` resuelve sin errores.
- TS strict base compartida (tsconfig raíz + extends).

## Notas
- Infra compartida · sin nativos · bloquea el resto.
